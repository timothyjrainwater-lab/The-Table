## WORK PACKET 1 — AoO Provocation Completeness (GAP-001)

### Objective

Implement and validate full 3.5e AoO provocation triggers for ranged attacks and spellcasting.

### Scope

**IN:**

* AoO provocation logic in `aidm/core/aoo.py`
* Interaction with `attack_resolver.py` and spellcasting intent
* Concentration checks triggered by AoO damage

**OUT:**

* Grapple-based AoO variants
* Exotic feats modifying AoO

### Artifacts Touched

* `aidm/core/aoo.py`
* `aidm/core/attack_resolver.py`
* Spell intent handling module
* Tests under `tests/core/test_aoo_*`

### Step Plan

1. Extract existing movement-based AoO trigger logic.
2. Define canonical provocation event list (RAW matrix).
3. Add ranged attack provocation path.
4. Add spellcasting provocation path.
5. Ensure damage-triggered concentration checks fire.
6. Validate interaction ordering with event log.
7. Add replay verification for new paths.
8. Update Known Tech Debt doc to close TD-005.

### Acceptance Criteria

* Casting in threatened square triggers AoO.
* Ranged attack in threatened square triggers AoO.
* Concentration check DC = 10 + damage works.
* Replay hash unchanged for prior test corpus.

### New Tests

* `test_aoo_spellcasting_provokes()` – proves casting in threat causes AoO.
* `test_aoo_ranged_provokes()` – proves ranged attack provokes.
* `test_concentration_after_aoo_damage()` – proves damage triggers check.

### Rollback Plan

Revert provocation additions; restore prior AoO movement-only behavior.

### Stop Conditions

* If provocation ordering breaks determinism (hash drift), halt.
* If spell intent cannot represent provoking action cleanly, escalate.
