# Citation Policy (D&D 3.5 Vault)

## Core rulebooks
Use abbreviations:
- **PHB** — Player’s Handbook
- **DMG** — Dungeon Master’s Guide
- **MM** — Monster Manual

Format:
- `PHB p.137–138`
- `DMG p.42`
- `MM p.12`

## Non-core sources
Use the **full title** of the source in the citation.

Format:
- `Book of Exalted Deeds p.12`
- `Complete Warrior p.54–55`
- `The Sunless Citadel p.3`

## Notes
- Always include `p.` and use an en dash for ranges.
- If a rule is cited from multiple sources, append additional citations; never overwrite.
